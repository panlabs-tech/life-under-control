# Vocabulário da interface

`CONTEXT.md` é a autoridade completa de domínio. Esta página é a referência curta para copy visual.

| Use | Significado na UI | Não use |
|---|---|---|
| Lar | unidade que possui todos os dados | workspace, tenant, conta, espaço |
| Pessoa | quem autentica e assina autoria | usuário, membro, perfil, papel |
| Área | domínio da vida, `ativa` ou `em breve` | categoria, seção |
| Painel | visão geral de largada | dashboard, home |
| Conta | regra recorrente que sabe quando, nunca quanto | despesa, boleto, assinatura, conta bancária |
| Lançamento | fato de um pagamento, com valor real, data e autoria | pagamento previsto, fatura, parcela |
| Competência | período ao qual o Lançamento pertence | mês de pagamento, vencimento |
| Agenda | projeção temporal de Contas e Tarefas | calendário |
| Indicação | candidata a experimentar e avaliar | favorito, wishlist, lista de desejos |
| Pontualidade | leitura derivada dos Lançamentos dentro da janela | campo persistido |

## Regras de copy

- UI em pt-BR; identificadores de código em inglês.
- Estados em minúsculas: `ativa`, `em breve`, `quitada`, `pendente`, `a vencer`, `encerrada`.
- Dinheiro: `R$ 1.234,56`.
- Datas: `dd/mm/aaaa`.
- Leituras derivadas são frases, não campos editáveis: “venceu há 8 dias”, “em 3 dias”, “13/14 no prazo”.
- Autoria diz “pago por Thiago” ou usa o chip da Pessoa; nunca implica propriedade privada ou permissão.
- Marca: “Life Under Control” e “LUC”. O codinome da opção visual nunca aparece na interface.
